Casting Curved Shadows on Curved Surfaces, Lance Williams, 1978


An OpenGL C++ implementation of basic shadow mapping using a depth buffer.


The project is tested on Linux and built with:

  • Programming language: C++
  • Graphics API: OpenGL
  • GLFW
  • GLEW
  • GLM
  • CMake
  • Open Asset Import Library (Assimp)

To build: cmake -G "CodeBlocks - Unix Makefiles" /path/to/CurvedShadows          To run: ./CurvedShadows

Camera Movements:

  • W,S,A,D,↑,↓ keys to move the camera front, back, left, right, up and down respectively
  • R to reset the camera
  • Left click and drag mouse to pan around
  • Scroll mouse to zoom
  • Q to quit
